##### **Description**:
Dhikr Daily is a project designed to help Muslims maintain their daily adhkars (remembrances of Allah) with ease and consistency. The app provides a user-friendly interface to track and count dhikrs, offers reminders for specific adhkars, and includes educational resources to deepen understanding. With features like a customizable adhkar bank, prayer-time-based background themes, and lecture videos, Dhikr Daily aims to make the practice of dhikr more accessible and engaging.

Inspired by the need for a simple yet effective tool to help Muslims stay consistent in their worship.

## Getting Started:

To run this project locally, you'll need to have Node.js and npm installed on your computer. Once you have these installed, follow these steps:

**1. Clone the repository:**

```
> git clone <repository-url>
```

**2. Navigate to the project directory using the command line.**

**3. Install dependencies:**

```
> npm install
```

**4. Start the Vite development server:**

```
> npm run dev
```

**5. Open your web browser and navigate to http://localhost:5173 (or the port shown in the terminal) to view the project.**

## Features:
This project includes:
- **Dhikr Counter**: A counter to track how many times a specific dhikr is recited.
- **Adhkar Bank**: A collection of different adhkars with the ability to add custom ones.
- **Prayer-Time Themes**: Background color changes based on the current prayer time.
- **Reminders**: Notifications for specific adhkars at designated times.
- **Educational Resources**: Lecture videos and articles about the importance and methods of dhikr.
- **Customizable Settings**: Adjust region-specific prayer times and add personal adhkars.
